## Install

```bash
yarn install
```

## Preview

```bash
yarn start
```

Visit https://localhost:8000

## Prettier

```bash
yarn format
```

## Test

```bash
yarn test
```

## Publish to npm

```bash
cd @antv/gatsby-theme-antv
npm publish
```
